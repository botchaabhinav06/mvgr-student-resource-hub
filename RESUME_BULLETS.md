# Resume Bullets

* Engineered a full-stack academic resource portal with role-based Student, Faculty, and Admin dashboards using React, TypeScript, Firebase Auth, Firestore, and Express.js.
* Architected a secure file storage system using Cloudflare R2 and backend-proxied signed URLs to protect sensitive academic content from public access.
* Implemented complex, state-based role authorization, ensuring students only access department-specific materials while faculty manage catalog workflows.
* Built comprehensive admin management utilities including real-time user roster management and automated database normalization tools to manage schema drift.
* Successfully deployed on Vercel and Render, implementing secure CI/CD pipelines to ensure production stability.
* Conducted rigorous security audits, resolving potential unauthorized file access vectors through rigorous backend token verification.
